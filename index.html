ADC

ADC is available on IO2 (PA6/ADC1_IN6), IO3 (PA4/ADC1_IN4), and IO4 (PA7/ADC1_IN7). 12-bit resolution, VREF+ = 3.3V. Single conversion mode: Start → Poll → Read → Stop per call.
adc_init(pin)

Initialize a 45-pin IO as an ADC analog input. Configures the GPIO to analog mode, sets up ADC1 with 480-cycle sampling time, single conversion, software trigger.

pininteger — pin number (2, 3, or 4)
Returns: 0 (ok), 1 (error — invalid pin or HAL failure)
Only one ADC channel can be active at a time. Calling adc_init with a different pin re-initializes the ADC for the new pin.
adc_init(3)  -- init IO3 (PA4) as ADC input
adc_read(pin)

Read the raw 12-bit ADC value (0–4095).

pininteger — pin number (2, 3, or 4)
Returns: integer 0–4095, or 0xFFFFFFFF on error
adc_init(2)
local raw = adc_read(2)
print("RAW: " .. raw)
adc_read_voltage(pin)

Read the ADC value converted to millivolts. Formula: raw * 3300 / 4095.

pininteger — pin number (2, 3, or 4)
Returns: integer mV, or 0xFFFFFFFF on error
adc_init(4)
local mv = adc_read_voltage(4)
print("Voltage: " .. mv .. " mV")
adc_deinit(pin)

De-initialize the ADC peripheral and disable its clock.

pininteger — pin number (2, 3, or 4)

PWM

PWM is available on IO14 (PB7 / TIM4_CH2) and IO15 (PB6 / TIM4_CH1). Timer base: APB1 42 MHz, prescaler calculated from target frequency.
t10_pwm_on(pin, freq, duty)

Start PWM output on a pin.

pininteger — pin number (14 or 15)
freqinteger — target frequency in Hz
dutyinteger — duty cycle percentage (0–100)
t10_pwm_on(14, 1000, 50)  -- 1 kHz, 50% duty on IO14
t10_pwm_off(pin)

Stop PWM output on a pin and de-initialize the timer.

pininteger — pin number (14 or 15)

Configure MIPI-DSI (serial) mode via RA8876 → SSD2828 serializer. All timing parameters are the same as DPI, plus lane count and video mode selection.

Parameters 1–13 are identical to t10_DPI

45-Pin Connector Pin Map

The 45-pin connector provides the following GPIO and special-function pins on the STM32F407VET6:

Pin #LabelSTM32 PinCapabilities
2UIO_2PA6GPIO, ADC1_IN6, TIM3_CH1
3UIO_3PA4GPIO, ADC1_IN4, SPI1_NSS
4UIO_4PA7GPIO, ADC1_IN7, TIM3_CH2, SPI1_MOSI
5UIO_5PC15GPIO
6UIO_6PE6GPIO
7UIO_7PE5GPIO
8UIO_8PD6GPIO
9UIO_9PD3GPIO
14UIO_14PB7GPIO, TIM4_CH2 (PWM)
15UIO_15PB6GPIO, TIM4_CH1 (PWM)
16UIO_16PC7GPIO
Pins 1, 10–13 are not connected to GPIO (GND, VCC, or unused). Only pins 2, 3, 4 have ADC capability. Only pins 14, 15 have PWM capability.
